# 携程数据迁移助手 (Ctrip Helper)

> Chrome 扩展程序，用于携程 VBK 后台的数据导出、导入和迁移

## 功能特性

### v1.0 - 当前版本

- ✅ **表单数据导出**：一键导出产品详情页的所有表单数据为 JSON
- ✅ **表单数据导入**：从 JSON 导入数据并自动填写表单
- ✅ **Ant Design 1.x 支持**：专门针对 Ant Design 1.x 组件优化
- ✅ **时间选择器填写**：通过点击时间面板来选择时间
- ✅ **繁简体自动转换**：自动处理语言字段的繁简体差异
- ✅ **搜索框智能匹配**：支持城市等搜索框的模糊匹配
- ✅ **null 值处理**：自动将 null 转换为空字符串

### v2.0 - 计划中

- 📋 **模版系统**：保存和应用数据模版
- 🔄 **通用表单引擎**：支持更多页面类型
- 🖼️ **图片批量下载**：一键下载页面图片
- 📊 **更多实用功能**

## 安装

### 从源码安装

1. 克隆仓库
```bash
git clone git@github.com:txz1006/ctirp-helper.git
cd ctirp-helper
```

2. 打开 Chrome 扩展管理页面
   - 访问 `chrome://extensions/`
   - 开启"开发者模式"

3. 加载扩展
   - 点击"加载已解压的扩展程序"
   - 选择项目目录

## 使用方法

### 1. 导出数据

1. 在携程 VBK 后台打开产品详情页
2. 点击页面右侧的扩展面板
3. 点击"导出数据"按钮
4. 复制生成的 JSON 数据

### 2. 导入数据

1. 在目标产品页面打开扩展面板
2. 将 JSON 数据粘贴到导入框
3. 点击"解析并转换"
4. 确认数据无误后点击"确认填写"

## 常见问题

### 1. 时间选择器填写失败？

时间选择器使用点击面板的方式填写，需要确保：
- 时间格式为 `HH:mm`（如 `18:00`）
- 时间面板能正常打开

### 2. 服务语言无法选中？

导入时会自动将简体转换为繁体（如"普通话" → "普通話"），如果仍然失败，请检查：
- 控制台是否有转换日志
- 下拉框选项是否与转换后的文本匹配

### 3. 搜索框找不到结果？

搜索框使用包含匹配策略：
- 会选择第一个包含关键词的选项
- 如果只有一个结果，直接选择
