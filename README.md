# 携程数据迁移助手 (Ctrip Helper)

> Chrome 扩展程序，用于携程 VBK 后台的数据导出、导入和迁移

## 功能特性

### v0.2.0 - 当前版本

- ✅ **表单数据导出**：一键导出产品详情页的所有表单数据为 JSON
- ✅ **表单数据导入**：从 JSON 导入数据并自动填写表单
- ✅ **模版系统**：保存和应用数据模版，快速复用表单配置
  - 💾 保存当前页面为模版（最多25个）
  - 📋 模版管理：预览、编辑、删除
  - 🔄 一键应用模版到新表单
- ✅ **Ant Design 1.x 支持**：专门针对 Ant Design 1.x 组件优化
- ✅ **时间选择器填写**：通过点击时间面板来选择时间
- ✅ **繁简体自动转换**：自动处理语言字段的繁简体差异
- ✅ **搜索框智能匹配**：支持城市等搜索框的模糊匹配
- ✅ **null 值处理**：自动将 null 转换为空字符串

### v1.0 - 已完成

- ✅ 基础导出/导入功能
- ✅ Ant Design 1.x 填写器
- ✅ 时间选择器面板点击填写
- ✅ 繁简体自动转换

## 安装

### 从源码安装

1. 克隆仓库
```bash
git clone git@github.com:txz1006/ctirp-helper.git
cd ctirp-helper
```

2. 打开 Chrome 扩展管理页面
   - 访问 `chrome://extensions/`
   - 开启"开发者模式"

3. 加载扩展
   - 点击"加载已解压的扩展程序"
   - 选择项目目录

## 使用方法

### 1. 导出数据

1. 在携程 VBK 后台打开产品详情页
2. 点击页面右侧的扩展面板
3. 点击"📤 导出数据"按钮
4. 复制生成的 JSON 数据

### 2. 保存为模版

1. 填写完表单后，点击页面底部"💾 保存为模版"按钮
2. 输入模版名称和描述（可选）
3. 点击保存
4. 最多可保存 25 个模版

### 3. 导入数据

**方式A：粘贴 JSON**
1. 在目标产品页面打开扩展面板
2. 选择"粘贴 JSON 数据"
3. 将 JSON 数据粘贴到导入框
4. 点击"下一步：解析并转换"
5. 确认数据无误后点击"确认填写"

**方式B：使用模版**
1. 在目标产品页面打开扩展面板
2. 选择"使用已保存的模版"
3. 从下拉框选择一个模版
4. 点击"下一步：解析并转换"
5. 确认数据无误后点击"确认填写"

### 4. 管理模版

1. 点击页面底部"📋 模版管理"按钮
2. 在面板中可以：
   - 👁 预览：查看模版包含的所有字段
   - ✏️ 编辑：修改模版名称和描述
   - 🗑️ 删除：删除不需要的模版

## 常见问题

### 1. 时间选择器填写失败？

时间选择器使用点击面板的方式填写，需要确保：
- 时间格式为 `HH:mm`（如 `18:00`）
- 时间面板能正常打开

### 2. 服务语言无法选中？

导入时会自动将简体转换为繁体（如"普通话" → "普通話"），如果仍然失败，请检查：
- 控制台是否有转换日志
- 下拉框选项是否与转换后的文本匹配

### 3. 搜索框找不到结果？

搜索框使用包含匹配策略：
- 会选择第一个包含关键词的选项
- 如果只有一个结果，直接选择
